# RateMyHousing
Created by Jonah Barnett, Aaron Brey, Kent Manion, and Andew Stappenbeck

use the following command too start server

```sh
uvicorn web:app --reload
```

To test backend use
```sh
python -m unittest src/Tests.py
```

To test webpage
```sh
python pytest
```
